# AI-Powered Assistance in Formulating Research Questions

An [Ethical Tech CoLab](https://github.com/Ethical-Tech-CoLab) project exploring how AI can help researchers move from a broad area of interest to a well-formed, answerable research question.

## Status

Early. The repository is a placeholder while scope is defined — a Next.js web app is planned but not yet scaffolded.

## Background

Formulating a good research question is one of the hardest and least-supported steps in the research process. A question that is too broad resists investigation; one that is too narrow yields findings nobody needs. This project asks where AI assistance genuinely helps at that stage, and where it risks flattening the researcher's own judgment.

## Goals

- Explore where AI assistance is useful in question formulation, and where it isn't.
- Build a tool that supports a researcher's thinking rather than substituting for it.
- Keep the researcher's reasoning visible and in their hands throughout.

## Planned stack

Next.js (TypeScript, App Router, Tailwind CSS), consistent with other Ethical Tech CoLab web projects.
